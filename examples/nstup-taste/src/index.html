What is new in v2.

A substantial 2026 rewrite of the default skill. The agent reads the brief, infers the right direction, and ships interfaces that do not look templated.

§0

Brief inference

The agent reads the room before generating: industry, audience, mood, motion depth, layout family.

§2

Design system map

When to reach for Material, shadcn, Tailwind, Radix, or native CSS. Framework-aware design decisions.

§5

Dark mode protocol

Dual-mode by default. Contrast and hierarchy parity across themes with automatic CSS variable generation.

§7

Block library

Contract for iterative block additions so the library stays coherent. Hero, feature grid, navigation schemas.

§9

Pre-flight check

Every checkbox must honestly pass before the agent ships output. No skipped validations.

§11

Redesign protocol

Audit-first on existing projects. Preservation rules and explicit modernisation levers.
