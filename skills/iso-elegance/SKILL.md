# Iso — L'Élégance Française

Design épuré, équilibré, avec une touche de raffinement français.

## Principes

1. **Le Vide est une Forme** — L'espace blanc n'est pas un vide, c'est un élément de design à part entière
2. **La Typographie est la Voix** — Une seule famille typographique, déclinée avec maîtrise
3. **La Couleur est un Accent** — Palette monochrome + une couleur d'accent (jamais plus)
4. **La Grille est un Rythme** — Ratios harmonieux (4, 8, 12, 16, 24, 32, 48, 64, 96, 128)

## Palette
- Fond : `#fafafa` (clair), `#111` (sombre)
- Texte : `#1a1a1a` (clair), `#f0f0f0` (sombre)
- Accent : une seule couleur, utilisée avec parcimonie
- Gris : en tints de la couleur de fond, pas de gris neutre

## Typographie
- Titres : satoshi, cabinet-grotesk, ou DM Sans
- Corps : Inter évité — utiliser Figtree, Outfit, ou Sora
- Échelle : `clamp(1rem, 0.8rem + 0.5vw, 1.25rem)` pour le corps

## Layout
- Marges généreuses (8-12% de chaque côté)
- Colonnes asymétriques (ratio 2:1, 3:2, jamais 1:1:1)
- Espacement : utilise exclusivement les valeurs de la grille
