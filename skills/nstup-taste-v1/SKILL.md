# Nstup Taste v1

Version originale du skill Nstup Taste. Utilise cette version si tu as besoin du comportement exact de v1 ou si la v2 est trop prescriptive.

## Différences avec la v2

La v1 ne contient PAS :
- Brief Inference (section 1 de la v2)
- Design System Map complet (section 4)
- Dark Mode Protocol (section 5)
- Redesign Protocol (section 6)
- Block Library (section 7)
- CSS Pattern Library (section 8)
- Pre-Flight Check (section 9)
- Règles françaises détaillées (section 10)
- Interdictions supplémentaires (section 11)

La v1 contient uniquement :
- Les 3 cadrans (VARIANCE / MOUVEMENT / DENSITÉ)
- Les règles anti-slop de base
- Les polices et couleurs interdites
- Les layouts interdits

Pour la version complète, installe `nstup-taste` (sans le suffixe `-v1`).
