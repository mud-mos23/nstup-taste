# Soft — Le Luxe Calme

Design premium qui respire la qualité sans crier. L'élégance discrète.

## Architecture
- **Double-Bezel** : un cadre subtil autour du viewport
- **Fluid Island** : navigation flottante intégrée au design
- **Magnetic Buttons** : boutons avec physique magnétique au hover

## Principes
- Contraste doux (ratio 4.5:1 minimum, viser 7:1 pour le confort)
- Typographie luxueuse (serif pour titres, sans-serif pour corps)
- Motion spring uniquement
- Ombres subtiles, flou contrôlé

## Palette
- Fond: #f5f3f0 (clair) / #121212 (sombre)
- Texte: #1a1a1a (clair) / #e0ddd5 (sombre)
- Accent: #8b7355 (or/brun)
- Secondaire: #4a6fa5 (bleu profond)
